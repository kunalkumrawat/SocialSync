## 🔄 Updating the Guidelines

### **To Deploy Updates:**

1. **Edit the HTML:**
   ```bash
   # Edit the local file
   nano docs/SocialSync_Brand_Guidelines.html
   ```

2. **Copy to deployment directory:**
   ```bash
   cp docs/SocialSync_Brand_Guidelines.html brand-guidelines-site/index.html
   ```

3. **Deploy to Vercel:**
   ```bash
   vercel --prod
   ```

### **Automatic Deployment (Optional)**
You can connect the GitHub repository to Vercel for automatic deployments:
1. Go to https://vercel.com/dashboard
2. Import your Git repository
3. Configure automatic deployments on push
