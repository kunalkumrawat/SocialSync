## 3. Logo System

### Primary Logo Lockup

The SocialSync logo consists of:
- Wordmark "SocialSync" in Poppins Black
- "by STAGE" attribution
- Tagline "Your Social Media Executive"

### Logo Specifications

**Minimum Sizes:**
- Digital: 120px width minimum
- Print: 25mm width minimum

**Clear Space:**
- Minimum space around logo = height of the "S" in SocialSync
- Never place other elements within this clear space

### Logo Variations

1. **Full Lockup** (Preferred)
   - SocialSync + by STAGE + tagline
   - Use for headers, marketing materials

2. **Short Lockup**
   - SocialSync + by STAGE (without tagline)
   - Use when space is limited

3. **Wordmark Only**
   - SocialSync (standalone)
   - Use for very tight spaces only

### Logo Don'ts

❌ **Don't:**
- Stretch or distort proportions
- Change colors or add custom colors
- Add effects (shadows, outlines, glows)
- Rotate or angle the logo
- Place on busy backgrounds
- Crop or rearrange elements
- Separate "by STAGE" from main wordmark

### Font Weight Guidelines

**Noto Sans:**
- **Regular (400):** Body text, descriptions, general content
- **Medium (500):** Emphasized text, file names, important labels
- **Bold (700):** Headings, button text, navigation, form labels
- **Black (900):** Stats numbers, critical information (use sparingly)

**Poppins:**
- **Semibold (600):** H3 headings, section subheadings
- **Bold (700):** H1, H2, major section titles
- **Black (900):** Hero text, display text, brand statements

### Letter Spacing

- **Headings:** -0.01em to -0.02em (tighter for large text)
- **Body Text:** 0 (normal)
- **Buttons:** 0.01em (slightly wider)

## 9. Motion & Animation

### Animation Principles

**Duration:**
- **Micro interactions:** 200ms (button hovers, icon changes)
- **Major transitions:** 300ms (navigation, view changes)
- **Page transitions:** 500ms (rare, significant changes)

**Easing:**
- `cubic-bezier(0.4, 0, 0.2, 1)` - Standard ease-in-out
- Creates smooth, natural motion

### Common Animations

#### Hover Effects
- **Buttons:** Scale 105%, duration 200ms
- **Icons:** Scale 110%, duration 300ms
- **Cards:** Scale 105%, shadow intensification, duration 300ms
- **Links:** Color transition, duration 200ms

#### Active States
- **Button press:** Scale 95%, duration 200ms
- **Navigation item:** Background gradient + scale 105%
- **Pulse indicator:** Fade opacity 50-100%, duration 1000ms, infinite

#### Loading States
- **Spinner:** Rotate 360deg, duration 1000ms, infinite
- **Skeleton:** Pulse opacity, duration 1500ms, infinite

### Animation Guidelines

**Do:**
- Use animations for feedback and state changes
- Keep animations subtle and purposeful
- Respect user's motion preferences (prefers-reduced-motion)

**Don't:**
- Overuse animations (causes distraction)
- Use very long durations (feels slow)
- Animate too many elements simultaneously

## 10. Voice & Messaging

### Brand Voice

SocialSync communicates with a voice that is:

**Professional**
- Reliable tool for serious creators and businesses
- Clear, accurate language
- Respects user's time and intelligence

**Approachable**
- Not stuffy or intimidating
- Conversational yet professional
- Helpful and friendly tone

**Confident**
- Direct and assured messaging
- We know what we're doing
- Not arrogant

**Action-Oriented**
- Active voice always
- Clear verbs guide next actions
- Specific and concrete language

### Writing Guidelines

#### Do's ✓
- Use **active voice:** "Connect your account" not "Your account can be connected"
- Be **specific:** "Post at 9:00 AM daily" not "Post regularly"
- Use **clear verbs:** Connect, Schedule, Post, Delete, Retry
- Keep it **concise:** Respect the user's time
- Provide **context:** Explain why something matters

#### Don'ts ✗
- Don't use jargon without explanation
- Don't be overly casual or use slang
- Don't blame the user for errors
- Don't use passive voice
- Don't be vague or ambiguous

### UI Copy Examples

#### Success Messages
```
✓ Account connected successfully!
✓ Post scheduled for 9:00 AM tomorrow
✓ Video posted to Instagram
```

#### Error Messages
```
✗ Unable to connect to Instagram
  Please check your credentials and try again.

✗ Failed to post video
  Video exceeds Instagram's 60-second limit. Trim and try again.

✗ Connection to Google Drive failed
  Check your internet connection and retry.
```

#### Warning Messages
```
⚠ Automated posting is paused
  Posts will not be published automatically. Enable in Settings to resume.

⚠ Low storage space
  Your Google Drive is nearly full. Free up space to continue syncing.
```

#### Empty States
```
No content yet
Connect your Google Drive to start managing your videos.

No posts scheduled
Add content to your queue to start scheduling posts.

No activity to show
Recent events and actions will appear here.
```

#### Button Labels
- **Primary actions:** Connect Account, Schedule Post, Save Settings
- **Secondary actions:** Cancel, Skip, Learn More
- **Destructive actions:** Delete, Remove, Disconnect

## 11. Accessibility

### WCAG 2.1 AA Compliance

SocialSync meets WCAG 2.1 Level AA standards for accessibility.

### Color Contrast Requirements

**Minimum Ratios:**
- Normal text (16px+): 4.5:1
- Large text (18px+ or 14px+ bold): 3:1
- UI components: 3:1

**Our Ratios:**
- White on Gray 800: 14.3:1 ✓ AAA
- Gray 300 on Gray 800: 7.1:1 ✓ AAA
- Success on Gray 800: 4.8:1 ✓ AA
- Error on Gray 800: 4.2:1 ✓ AA
- Warning on Gray 800: 7.1:1 ✓ AAA
- Info on Gray 800: 4.5:1 ✓ AA

### Keyboard Navigation

- All interactive elements keyboard accessible
- Logical tab order
- Visible focus states
- Keyboard shortcuts for common actions
- Escape key closes modals/dropdowns

### Screen Reader Support

- All icons paired with text labels or ARIA labels
- Status updates announced
- Form fields have associated labels
- Error messages linked to fields
- Loading states announced

### Best Practices

**Do:**
- Provide text alternatives for all icons
- Maintain high contrast ratios
- Use semantic HTML
- Test with keyboard only
- Test with screen readers
- Respect motion preferences

**Don't:**
- Use color as only indicator
- Disable zoom
- Remove focus outlines
- Use very small text (<12px)
- Create keyboard traps
- Auto-play videos with sound

### Testing Checklist

- [ ] Test with keyboard only (no mouse)
- [ ] Test with screen reader (VoiceOver, NVDA)
- [ ] Check color contrast with automated tools
- [ ] Zoom to 200% and verify functionality
- [ ] Test with Windows High Contrast Mode
- [ ] Verify focus indicators visible
- [ ] Check form validation messages
- [ ] Test with motion reduced (prefers-reduced-motion)

### Resources

- **WCAG Guidelines:** https://www.w3.org/WAI/WCAG21/quickref/
- **Contrast Checker:** https://webaim.org/resources/contrastchecker/
- **Screen Readers:** VoiceOver (macOS), NVDA (Windows)
- **Browser Extensions:** axe DevTools, WAVE
