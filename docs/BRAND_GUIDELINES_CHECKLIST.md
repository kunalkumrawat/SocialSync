## 💡 Tips While Working

**Take breaks:**
- Work in 2-hour sessions
- Take 15-minute breaks
- Fresh eyes catch mistakes

**Save often:**
- Figma auto-saves, but export backups
- Version your file (v1.0, v1.1, etc.)
- Keep original screenshots safe

**Get feedback:**
- Show work-in-progress to team
- Ask for input on colors/layouts
- Iterate based on feedback

**Stay organized:**
- Name all layers clearly
- Use folders/groups
- Keep assets organized
- Document decisions
